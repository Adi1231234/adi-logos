# Logo Studio

A small React/Vite app for designing and exporting chat-bubble logo variants.

## Features

- Edit bubble shape, color, border, and horizontal mirror.
- Add folder or image/document inner icons.
- Save named logo presets.
- Export PNG or SVG, including ZIP export for all presets.
- Choose transparent or solid export backgrounds.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
